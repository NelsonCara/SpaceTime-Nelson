  ## 🖥️ Projeto
Esse é um projeto web responsivo de uma cápsula do tempo para exibir memórias em uma linha do tempo.

## 🚀 Tecnologias
Este projeto foi desenvolvido durante a NWL da Rocketseat com as seguintes tecnologias:

- HTML
- CSS
- Git e Github
